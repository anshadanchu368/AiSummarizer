# AI Summarizer Generator

This project is an AI Summarizer Generator that utilizes RapidAPI's services to summarize articles based on their URLs. It is built with React, Redux Toolkit for state management, and Tailwind CSS for styling.

## Features

- Enter a URL and generate a summary of the article.
- Keeps track of previously summarized articles.
- Copy the article URL with a single click.
- Utilizes RapidAPI's services for article extraction and summarization.
- Built with React, Redux Toolkit, and Tailwind CSS.

## Technologies Used

- React
- Redux Toolkit
- Tailwind CSS
- RapidAPI
